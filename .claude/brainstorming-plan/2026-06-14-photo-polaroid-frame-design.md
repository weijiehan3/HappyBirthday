# 照片拍立得边框设计

## 背景

生日照片直接展示在页面中会显得生硬，且生日帽在当前人物照片上略微偏上。用户选择了“温柔拍立得生日卡”方向，希望保留现有动画和照片主体，同时增强照片出现时的仪式感。

## 方案

在 `.six` 照片场景中保留现有 `.photo-stage`，新增 `.photo-print` 作为照片和帽子的共同定位坐标系。`.photo-stage` 负责外层拍立得卡片视觉，包括暖白背景、浅粉金边框、轻阴影、轻微旋转和底部留白；`.photo-print` 保持相对定位，确保帽子坐标仍然基于照片本体，而不会被拍立得底部留白影响。

生日帽只做小幅下移，不改变动画选择器和出现节奏。照片主体、帽子和祝福文字仍沿用原来的数据流：`customize.json` 修改 `imagePath`，`script/main.js` 使用 `[data-node-name="imagePath"]` 替换图片地址。

## 验收

- 照片应呈现为柔和拍立得卡片，不遮挡人物和祝福文字。
- 帽子应更贴近头顶，不再明显悬空。
- 桌面和移动端下帽子位置应跟随照片缩放。
- `npm run build` 应正常通过。
